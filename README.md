# abe-sz

Portfolio website source and planning materials for Abe Sz.

This repo is starting foundation-first: capture the portfolio thesis, content architecture, project framing, and operating principles before choosing the site implementation details.

Current foundation docs live in `docs/foundation/`.

## Development

```bash
npm install
npm run dev
```

The dev server binds to `0.0.0.0` so the site can be viewed from other devices
on the same Tailscale network, for example `http://gbabe:5173` or
`http://100.99.99.99:5173`.

```bash
npm run build
```
