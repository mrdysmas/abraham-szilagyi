# Work Media Shelf

Store only public-safe portfolio media here. Do not add real customer data or private identifiers.

Expected proof labels:

- Demo-safe screenshot
- Prototype recording
- Sanitized workflow diagram

Public-safe policy:

- No customer data.
- No private addresses, emails, job IDs, asset IDs, tenant names, environment names, API URLs, tokens, or exact coordinates unless intentionally public.
- Prefer mocked, redacted, or recreated visuals over source production captures.

Shelf layout:

- `demo-safe-screenshots/`
- `prototype-recordings/`
- `sanitized-workflow-diagrams/`
